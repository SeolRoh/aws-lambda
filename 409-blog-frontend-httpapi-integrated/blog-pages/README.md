# 409-blog-frontend-apigw-integrated-blog-pages

블로그 예제의 프론트엔드 구현. `407-blog-frontend-router` 예제와 완전히 동일하다.

## 환경

- Node.js v14
- React 18
- React Router 6

## 로컬 실행

```bash
$ npm start
```

로컬 서버를 연결하기 위해 `blog-api` 프로젝트의 로컬 서버를 기동한다.

## 빌드

```bash
$ npm run build
```

빌드 결과물은 `blog-api`에 의해 통합되어 함께 제공된다.
